VaporBoy Github Repo

Aaron Turner (torch2424) built VaporBoy app as an Open Source app. This SERVICE is provided by Aaron Turner at no cost and is intended for use as is. This page is used to inform website visitors regarding my policies with the collection, use, and disclosure of Personal Information if anyone decided to use my Service. This application offers reading files from file sharing (Google Drive) services. This application does not store this information anywhere than inside of your web browser, and has the sole purpose of only downloading a file from Google Drive, and using it within the emulator. This application uses Google Analytics for performance monitoring, and to the best of my knowledge, does not send any personally identifable information (I didn't change much of the default settings). This project is open source, and contributors will strive to violate anyone's privacy. Please feel free to read the source at anytime, and open issues.
